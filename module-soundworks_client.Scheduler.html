Class: Scheduler

soundworks/client.Scheduler

Interface for the client 'scheduler' service.

This service provides a scheduler synchronised among all client using the sync service. It internally uses the scheduler provided by the wavesjs library.

While this service has no direct server counterpart, it's dependency to the sync service requires the existance of a server. Also, the service requires a device with WebAudio ability.
