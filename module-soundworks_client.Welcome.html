Class: Welcome

soundworks/client.Welcome

Interface of the client 'welcome' service.

This service is used to create the home page of a soundworks application. It also works in conjunction with the platform service to execute the hooks provided to properly initialize requirements of the application (for example: use the user touch event to enter the application to initialize a proper AudioContext).

If any of the application's requirements fail, an error message is displayed instead of the home page.
