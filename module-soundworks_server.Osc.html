Class: Osc

soundworks/server.Osc

Interface of the server 'osc' service.

This server-only service provides support for OSC communications with an extenal software (e.g. Max). The configuration of the service (url and port) should be defined in the server configuration, cf. module:soundworks/server.envConfig.

The service currently only supports UDP protocol.

Example
// inside the experience constructor
this.osc = this.require('osc');
// when the experience has started, listen for incomming message
this.osc.receive('/osc/channel1', (values) => {
  // do something with `values`
});
// send a message
this.osc.send('/osc/channel2', [0.618, true]);
