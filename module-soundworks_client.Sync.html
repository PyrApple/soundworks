Class: Sync

soundworks/client.Sync

Interface of the client 'sync' service.

This service synchronize the local audio clock of the client with the clock of the server (master clock). It then internally relies on the WebAudio clock and requires the platform to access this feature.

The service must be used with its server-side counterpart

Note: the service is based on github.com/collective-soundworks/sync.

// when the experience has started, translate the sync time in local time
const syncTime = this.sync.getSyncTime();
const localTime = this.sync.getAudioTime(syncTime);
