Class: CanvasView

soundworks/client.CanvasView

View designed for experiences where 2d graphical rendering is needed. The view is basically a SegmentedView with a canvas element in the background.

new CanvasView(template, content, events, options)

WARNING Views should preferably by created using the Experience#createView method.
