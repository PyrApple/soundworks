Class: Platform

soundworks/client.Platform

Interface for the client 'platform' service.

This services is responsible to give general informations about the user's device (cf. client.device) as well as check availability and provide hooks to initialize the features required by the application (audio, microphone, etc.). If one of the required definition is not available, an view is created with an error message and the client.compatible attribute is set to false.

Available built-in definitions are:

  • 'web-audio'
  • 'mobile-device'
  • 'audio-input'

Most of these feature requiring an interaction or a confirmation from the user in order to be initialized correctly, this service should be used in conjonction with the welcome service.
