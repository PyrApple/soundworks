Class: MotionInput

soundworks/client.MotionInput

Interface of the client 'motion-input' service.

This service provides a wrapper for the motionInput external module.

Constructor

new MotionInput()

WARNING This class should never be instanciated manually

Source:
  • client/services/MotionInput.js, line 28
Example
// in the experince constructor
this.motionInput = this.require('motioninput', { descriptors: ['energy'] });
// when the experience has started
if (this.motionInput.isAvailable('energy')) {
  this.motionInput.addListener('energy', (data) => {
    // digest motion data
  });
} else {
  // handle error
}
