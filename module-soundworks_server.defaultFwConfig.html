Namespace: defaultFwConfig

soundworks/server.defaultFwConfig

Configuration parameters of the Soundworks framework. These parameters allow for configuring server-side components of the framework such as Express and SocketIO.

If the application is created using the provided application template, most of these values should not be changed.
